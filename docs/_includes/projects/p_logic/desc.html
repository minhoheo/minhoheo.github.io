Environment

Chain REAction Tool Environment (CREATE) is based on the popular puzzle The Incredible Machine. The objective is to solve Physics-based puzzles by using an available set of tools, carefully performing their selection and positioning. The common underlying objective is to make the red ball (target) reach the goal (green) which may or may not be moving.

Functionalities in the environment include:
  • Multi-step environment: Agent places a tool after every few iterations of simulation acting on image observations.
  • Gym interface: Enables convenient training of reinforcement learning algorithms.
  • Simple and Fast: Runs headless and supports flexible resolutions for rendering.
  • Tools: 12 base tool types, with many more variations of sizes, angles, friction parameters.
  • Tasks: 12 complex tasks with stochastic variations and an easy interface to create many more tasks. Suitable for meta reinforcement learning.
